# Magnifica Humanitas

Lector web (modo oscuro) de la Carta Encíclica **_Magnifica Humanitas_** del Papa León XIV, sobre la custodia de la persona humana en el tiempo de la inteligencia artificial (2026).

🌐 **Sitio en vivo:** https://magnificahumanitas.live

Cada página ofrece selector de idioma, buscador, tabla de contenidos, ajuste de tipografía/ancho y exportación a PDF, con una presentación de lectura optimizada.
